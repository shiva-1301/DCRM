User Guide

Complete guide to using the DCRM System

Getting Started

  1. Log in to the DCRM System using your credentials
  2. Navigate to the Dashboard to upload CSV files for analysis
  3. Upload your DCRM measurement files in CSV format
  4. Click "Predict" to get fault classification results
  5. Validate predictions to improve model accuracy

File Upload & Prediction

  1. Upload File: Drag and drop or click "Browse Files" to select your CSV file
  2. File Format: Ensure your CSV has Channel-1 columns (Coil Current C1, Contact Travel T1, DCRM Res CH1, DCRM Current CH1)
  3. Predict: Click the "Predict" button to analyze the file
  4. Review Results: View the predicted fault type and confidence levels
  5. Validate: Click "Correct" if accurate, or "Incorrect" to provide the correct label

SOS Request Feature

  1. Navigate to "SOS Request" in the sidebar
  2. Select the problem type from the dropdown menu
  3. Provide a detailed description of the issue
  4. Click "Send SOS Request" to notify your administrator
  5. Track the status of your requests in the SOS history section

Field Advisor Chatbot

  1. Navigate to "Field Advisor" in the sidebar
  2. Select any text from the website by highlighting it
  3. Type your question in the chat input field
  4. The chatbot will provide helpful answers about DCRM system, fault types, and usage
  5. You can ask about: healthy readings, main contact faults, arc faults, file uploads, predictions, and more

Employee Interactions

  1. Navigate to "Interactions" in the sidebar
  2. View your conversation list on the left side
  3. Click on a conversation to open the chat
  4. Type your message and press Enter or click Send
  5. Communicate with other employees under the same admin

Viewing History

  1. Navigate to "My History" to view all your predictions
  2. See details of each prediction including filename, fault type, and timestamp
  3. Track your validation activities and model improvements

Admin Features (Admin Only)

  1. Access the "Admin Panel" to manage employees
  2. View all SOS requests in "SOS Management"
  3. Resolve SOS requests by clicking the "Resolve" button
  4. Add new employees and manage user accounts
  5. View predictions from all employees

Tips & Best Practices

  • Always validate predictions to improve model accuracy
  • Use descriptive problem descriptions when sending SOS requests
  • Check the Field Advisor chatbot for quick answers to common questions
  • Keep your CSV files properly formatted with Channel-1 data
  • Regularly review your prediction history to track patterns
  • Use employee interactions for collaboration and knowledge sharing
